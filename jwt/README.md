# jwt

This folder contains examples of Json Web Tokens (JWTs).

Standard: <https://datatracker.ietf.org/doc/html/rfc7519>
jwt.io: <https://jwt.io/>

## Scripts

```bash
cd scripts
node index.js
```

## Server example

```bash
cd server
node index.js
```

The command above starts the server. Use Postman to create requests.
